# 🧠 Data Science Cheatsheet: MiniCourse Edition

This is a hands-on Python-based mini-course designed for self-learners, analysts, and aspiring data scientists. Every topic is a folder containing code examples + brief explanations.

---

## 📁 Modules

| No. | Module | Description |
|-----|--------|-------------|
| 1 | [Python Fundamentals](./1_python_fundamentals) | Syntax, functions, control flow, and built-ins |
| 2 | [Data Manipulation](./2_data_manipulation) | NumPy, Pandas, cleaning, transformation |
| 3 | [Data Visualization](./3_data_visualization) | Matplotlib, Seaborn, plotting examples |
| 4 | [Statistics & ML](./4_statistics_ml) | Descriptive stats, ML pipelines (WIP) |

---

## ✅ How to Use

- Clone this repo or download the folder.
- Open any `.py` file in VS Code or Jupyter.
- Run the examples — all code is real and executable.

> This repo is 100% beginner-friendly and designed for revisiting core concepts quickly.
